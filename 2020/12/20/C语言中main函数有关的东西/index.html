3. C语言的返回值

返回值返回给哪里去了呢？？
答案是返回给操作系统了，表示程序退出的状态

如果返回是0，那么表示程序正常退出
这个返回值是可以自己定义的
在Linux系统中$?是上一个命令的退出状态

例如我编译并执行下面这段程序：

然后再在shell中输入：
$?
那么会输出：
1: command not found

或者直接echo "$?"
那么会输出 1
同理，return 0会使得$?变成0

而在Windows 操作系统中，假如说用cmd 执行了刚刚编译出来的程序，那么可以通过：
echo %ERRORLEVEL%
来查看返回值
